# Safety And Scientific Integrity

## Implemented Controls

- The JSON executor calls only tools in a fixed registry and selected profile
  allowlist.
- Planner output is JSON data; it is never executed as shell commands.
- Agent instructions prohibit invented energies, structures, files, and
  calculation results.
- Verification checks reported paths, ORCA/xTB expected output files, and
  numeric energy-like values before reporting execution as verified.
- Result-query tools return "Property not found" when a value is absent.
- Local xTB and ORCA executable paths are selected through application
  configuration in `.env.cspilot`, not generated by an LLM.

## External Software

Calculations execute third-party software on local input. Users remain
responsible for:

- installing and licensing ORCA correctly;
- choosing methods, basis sets, charges, multiplicities, and models suitable
  for the scientific question;
- reviewing raw output before relying on reported properties.

## Planned Multiwfn Boundary

Multiwfn is not currently implemented. When it is added, only reviewed,
whitelisted Multiwfn scripts or fixed operations should be exposed to an
agent. Arbitrary LLM-generated Multiwfn or shell scripts must not be executed.

## Secrets

Keep `AGAPI_API_KEY` in `.env.cspilot` outside version control. Do not include
credentials in reports, example inputs, or issue logs.
